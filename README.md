# Founder's Note - AI-Powered Executive Brief

Your personal AI assistant that transforms daily chaos into structured clarity, strategic insight, and actionable guidance.

## üéØ What This Is

Founder's Note is an AI-powered daily executive brief system for startup founders. It:

- **Organizes your day** with intelligent time blocks and energy management
- **Surfaces strategic insights** from your calendar, tasks, and metrics
- **Guides decisions** with frameworks and AI coaching
- **Detects patterns** in your behavior and suggests improvements
- **Functions as a mentor** providing daily coaching prompts and strategic guidance

## üèóÔ∏è Tech Stack

- **Frontend**: Next.js 14 (App Router) + TypeScript + Tailwind CSS
- **Backend**: Next.js API Routes
- **Database**: PostgreSQL (with Prisma ORM)
- **Vector DB**: pgvector (for semantic search)
- **Authentication**: Clerk
- **AI**: OpenAI GPT-4 + Embeddings
- **Hosting**: Ready for Vercel + Supabase
